## 🗂 Como baixar o projeto

```bash

    # Clonar o repositório
    $ git clone https://github.com/edpedro/NevControl.git

    # Entrar no diretório
    $ cd web
    $ cd server

    # Instalar as dependências
    $ yarn install

    # Iniciar o projeto
    $ cd web - yarn start
    $ cd server - yarn dev
```
