## Como baixar o projeto

```bash
#Clona repositório
$ git clone https://github.com/edpedro/NevControl.git

#Entra no diretório
$ cd web

#Entra nas pasta Frontend e instalar dependência
$ yarn

#Inicia o projeto
$ yarn start

```
