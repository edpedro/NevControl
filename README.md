## Como baixar o projeto

```bash
#Clona repositório
$ git clone https://github.com/edpedro/NevControl.git

#Entra nos diretórios
$ cd NevControl

#Entra nas pasta web e server, instalar dependência
$ yarn

#Inicia o projeto
$ yarn start - web
$ yarn dev - server

```
